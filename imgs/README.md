# Pasta de Imagens (imgs)

Esta pasta armazena todas as imagens utilizadas na documentação do projeto, incluindo:

- diagramas do modelo dimensional (esquema estrela)
- arquitetura do pipeline (Bronze → Silver → Gold)
- gráficos gerados nas análises (taxas de attrition por segmento)
- tabelas, prints de resultados ou evidências necessárias para o MVP

Nenhuma imagem contém dados sensíveis.  
Todas as imagens servem exclusivamente para fins de documentação.
